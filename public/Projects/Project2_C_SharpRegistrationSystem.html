New Employee Registration Appliction

This C# application was developed for University during the 03 programming module. Developed closely with my coursemate Connor Gill-Mead.

The system utilises C#, .NET, and SQLite, as well as Software engineering principles as well as having full CRUD opperations.

  • Enables new employees to set up an account with their details.
  • Enales users to read their information and download it.
  • Enables users to print their data.

Admin-Only Features

  • Enabled super users to search, order, filter and remove users data from the SQLite Database.
  • The system has full regular regression for full data validation on new user creation.

General System features

  • Darkmode switch that saves to local memory.
  • Xunit testing across the whole codebase using both unit and integration testing to cover all methods implemented.
